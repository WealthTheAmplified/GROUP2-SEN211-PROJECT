Introduction To Function

A function is a block of code that performs a specific task and can be called multiple times from different parts of a program. Functions are used to organize code, reduce repetition, and improve modularity.


Characteristics of Fuctions:

  • A function has a unique name that identifies it.
  • A function takes arguments or parameters that are passed to it when it is called.
  • A function returns a value to the calling program.
  • A function can be called multiple times from different parts of a program.

Types of Functions

  1. Library functions: Pre-defined functions that are part of the programming language's standard library.
  2. User-defined functions: Functions that are defined by the programmer.

Declaring Functions

  • The syntax for declaring a function is: return_type function_name(parameter_list) { function_body }
  • The return_type specifies the type of value the function returns.
  • The function_name is the unique name of the function.

Function Calling

  • The syntax for calling a function is: function_name(argument_list);
  • The argument_list specifies the values to be passed to the function.

Advantages of Functions

  • Code reusability: Functions can be called multiple times from different parts of a program.
  • Code modularity: Functions break down a large program into smaller, manageable modules.
  • Easier maintenance: Functions make it easier to modify or update code.

Examples:

Example of a Function:

int add(int x, int y) {

return x + y;

}

int main() {

int result = add(5, 3);

printf("%d", result); // prints 8

}


Example of a Library Function:

printf("Hello, World!");

scanf("%d", &num);

strlen("Hello"); // returns 5

rand(); // generates a random number

exit(0); // exits the program


Example of a User-Defined Function:

void greet() {

printf("Hello, World!");

}

int main() {

greet(); // prints Hello, World!

}


Example of a Function with Arguments:

int multiply(int x, int y) {

return x * y;

}

int main() {

int result = multiply(5, 3);

printf("%d", result); // prints 15

}

Example of a Function with Return Value:

int add(int x, int y) {

return x + y;

}

int main() {

int result = add(5, 3);

printf("%d", result); // prints 8

}


Example of a Function with Multiple Return Statements:

int max(int x, int y) {

if (x > y) {

return x;

} else {

return y;

}

}

int main() {

int result = max(5, 3);

printf("%d", result); // prints 5

}


Example of a Recursive Function:

int factorial(int n) {

if (n == 0) {

return 1;

} else {

return n * factorial(n - 1);

}

}

int main() {

int result = factorial(5);

printf("%d", result); // prints 120

}


Example of a Function with Default Arguments:

int power(int x, int y = 2) {

int result = 1;

for (int i = 0; i < y; i++) {

result *= x;

}

return result;

}

int main() {

int result1 = power(2);

int result2 = power(2, 3);

printf("%d", result1); // prints 4

printf("%d", result2); // prints 8

}


Example of a Function with Variable Arguments:

int sum(int count, ...) {

va_list args;

va_start(args, count);

int result = 0;

for (int i = 0; i < count; i++) {

result += va_arg(args, int);

}

va_end(args);

return result;

}

int main() {

int result1 = sum(3, 1, 2, 3);

int result2 = sum(5, 1, 2, 3, 4, 5);

printf("%d", result1); // prints 6

printf("%d", result2); // prints 15

}
