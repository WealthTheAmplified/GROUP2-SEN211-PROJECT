Keywords, Variables and Assignment Statements

Keywords are reserved words in a programming lauguage that have special meanings.
They cannot be used as variable names. Examples of keywords include if, else, while,
for, int, and double. Keywords are used to define the structure and syntax of a program.
Variable are names given to memory locations to store data. A variable has a data type,
which determines the type of data that can be stored in it. Variables can be assigned a
new value. There are several type of variables, including integer variables, floating-point
variables, and character variables.

Types of Variables:

  • Integer variables are used to store whole numbers. They are declared using the int keyword.
    Example: int x = 5;
  • Floating-point variables are used to store decimal numbers. They are declared using the
    double keyword. Example double y = 3.14
  • Character variables are used to store single characters. They are declared using the character
    keyword. Example: character z = 'A';

Assignment Statement

Assignment statements are used to assign a value to a variable. The syntax for an assignment
statement is a variable_name = value;. For example: x = 5; assigns the value 5 to the variable x.


Rules of Naming Variables:

Variable names must start with a letter or underscore. They can contain letters, digits and underscore.
Variable names cannot contain special characters or keywords. Variable names must be unique within
the program. Examples of valid variable names include x, y, z, and total marks. Examples of invalid
variable names include 123, if total-marks.
